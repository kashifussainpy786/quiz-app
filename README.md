# quiz-app

"A Python CLI quiz app that loads questions from JSON, tracks score, and saves results."

## Features
- Questions loaded from JSON file
- Multiple choice answers (A, B, C, D)
- Score tracking
- Results saved to JSON file
- Game over on wrong answer

## Technologies Used
- Python 3
- JSON (data storage)
- OOP (Object-Oriented Programming)

## How to Run
1. Make sure Python 3 is installed
2. Clone or download this repo
3. Add questions to quiz.json file
4. Run: `python quiz.py`

## Concepts Applied
- Classes and Objects
- File Handling
- Error Handling
- JSON Integration
- CLI Interface
